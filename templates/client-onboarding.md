# Client Onboarding Checklist

> Standard onboarding workflow for new Marketing Powered clients. This checklist ensures all required information is gathered and all systems are configured before campaign work begins.

---

## Phase 1: Discovery & Information Gathering

### Client Information
- [ ] Client name, legal entity, and primary contact
- [ ] Industry and vertical classification
- [ ] Website URL and all digital properties
- [ ] Social media accounts (all platforms)
- [ ] Google Business Profile (if applicable)
- [ ] Current marketing vendor list (agencies, freelancers, tools)

### Business Context
- [ ] Business model and revenue streams
- [ ] Target markets (geographic, demographic)
- [ ] Primary products/services to promote
- [ ] Average deal size / customer lifetime value
- [ ] Sales cycle length and process
- [ ] Current lead volume and conversion rates

### Brand Assets
- [ ] Logo files (SVG, PNG, various formats)
- [ ] Brand style guide (if exists)
- [ ] Color palette (hex codes)
- [ ] Typography / font specifications
- [ ] Image library / approved photography
- [ ] Brand voice guidelines
- [ ] Compliance/legal review requirements

### Historical Data
- [ ] Previous campaign performance data (last 6-12 months)
- [ ] Google Analytics access (GA4 property)
- [ ] Ad platform historical data (Meta, Google, etc.)
- [ ] Email marketing data (list size, open rates, conversion rates)
- [ ] CRM data (lead sources, conversion by channel)
- [ ] Previous content audit (what exists, what performed)

### Competitive Landscape
- [ ] Top 3-5 direct competitors identified
- [ ] Competitive positioning / differentiation documented
- [ ] Known competitor marketing activities
- [ ] Market share estimates (if available)

---

## Phase 2: Access & Platform Setup

### Advertising Platforms
- [ ] Meta Business Manager — partner access granted
- [ ] Meta Ad Account — advertiser access granted
- [ ] Meta Pixel — verified and firing
- [ ] Google Ads account — MCC access granted
- [ ] Google Analytics 4 — editor access granted
- [ ] Google Tag Manager — access granted
- [ ] Google Search Console — access granted
- [ ] TikTok Ads Manager — access granted (if applicable)
- [ ] LinkedIn Campaign Manager — access granted (if applicable)
- [ ] Pinterest Ads — access granted (if applicable)
- [ ] X (Twitter) Ads — access granted (if applicable)

### Website & Landing Pages
- [ ] Website CMS access (if needed for landing pages)
- [ ] Hosting information and deployment process
- [ ] DNS access (if needed for subdomain setup)
- [ ] SSL certificate verified for all domains
- [ ] Tracking pixel installation verified across all pages

### Communication & Reporting
- [ ] Client communication channel established (email, Slack, etc.)
- [ ] Reporting frequency agreed (weekly, bi-weekly, monthly)
- [ ] Report delivery format agreed (email, dashboard, presentation)
- [ ] Meeting cadence established (weekly check-in, monthly review, QBR)
- [ ] Escalation contact for urgent issues

---

## Phase 3: System Configuration

### Agent 18 - Knowledge Base Setup
- [ ] Client profile created in knowledge base
- [ ] Brand guidelines indexed
- [ ] Historical performance data loaded
- [ ] Competitor profiles initialized
- [ ] Client preferences documented (communication style, priorities, sensitivities)

### Agent 15 - Workflow Configuration
- [ ] Client project created in project management system
- [ ] SLA timelines configured per engagement agreement
- [ ] Recurring task schedules set up (reporting, optimization cycles)
- [ ] Team notifications configured
- [ ] Quality checkpoint thresholds set

### Agent 17 - Budget Configuration
- [ ] Client budget loaded into tracking system
- [ ] Budget allocation plan configured
- [ ] Spend alert thresholds set
- [ ] Billing cycle and invoicing schedule documented
- [ ] Budget pacing baseline established

### Agent 13 - Analytics Setup
- [ ] Cross-channel dashboard created
- [ ] KPI targets configured with alert thresholds
- [ ] Attribution model selected and configured
- [ ] Automated reporting schedule set
- [ ] Baseline performance metrics documented

---

## Phase 4: Strategy Development

### Agent 01 - Competitive Intelligence
- [ ] Initial competitive analysis completed
- [ ] Competitor ad library audit completed
- [ ] Competitor landing page analysis completed
- [ ] Competitive positioning map created
- [ ] Opportunity gaps identified

### Agent 02 - Campaign Strategy
- [ ] Client brand analysis completed
- [ ] Audience personas developed (minimum 3)
- [ ] Multi-channel campaign architecture designed
- [ ] Budget allocation model created
- [ ] Campaign proposal presented and approved
- [ ] Creative briefs generated for all channels

---

## Phase 5: Launch Preparation

### Content & Creative
- [ ] Initial content strategy and editorial calendar developed (Agent 03)
- [ ] First authority content piece produced (Agent 04)
- [ ] Ad creatives produced for all channels (Agent 05)
- [ ] Landing pages built and tested (Agent 06)

### Campaign Setup
- [ ] All platform campaigns built in draft mode (Agents 07-09)
- [ ] Tracking verified across all touchpoints
- [ ] A/B test variants prepared
- [ ] Launch checklist completed (Agent 15)

### Final Approval
- [ ] Campaign proposal approved by client
- [ ] Creative assets approved by client
- [ ] Landing pages approved by client
- [ ] Budget confirmed by client
- [ ] Launch date confirmed
